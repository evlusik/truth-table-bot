# truth-table-bot
Бот для вычислений логических выражений, построения таблицы истинности


## Функционал

- Проверка правильности логических выражений
- Построение таблиц истинности
- Подсчет количества истинных/ложных значений
- Экспорт в TXT и CSV
- Удобная клавиатура в Telegram

## Доступные символы

| Операция | Символ | Пример |
|----------|--------|--------|
| И (AND) | `&` | `a & b` |
| ИЛИ (OR) | `|` | `a | b` |
| НЕ (NOT) | `!` | `!a` |
| Импликация | `→` | `a → b` |
| Эквивалентность | `↔` | `a ↔ b` |
| Скобки | `( )` | `(a & b) \| c` |

**Переменные:** любые латинские буквы (кроме V, A, B, C, D, E, F, G, H, X)

## Примеры выражений
a & b
!(a | b) → c
(p → q) & !r
